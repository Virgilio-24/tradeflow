Como funciona

Três passos para importar

Do URL do produto ao teu catálogo, tudo automático.

1

Cria a tua conta

Escolhe um plano, introduz o email e o domínio da tua loja. Recebes uma license key de imediato.

2

Envia o URL do produto

Faz um POST /scrape com o URL da Shein, Zara ou outra loja suportada. Tens um job ID de volta em milissegundos.

3

Recebe os dados estruturados

Nome, preço, imagens, tamanhos, cores e stock — tudo em JSON, pronto a inserir no teu catálogo.

API REST

Integra em minutos

Dois headers, um POST, e tens o produto. Funciona com qualquer linguagem ou framework.

  • ✓ Resposta assíncrona via job ID — sem timeouts
  • ✓ Retorna nome, preço, imagens, tamanhos e stock
  • ✓ Endpoint /usage para controlar o consumo
  • ✓ Erros claros: 401, 402, 403, 429, 422
Ver documentação completa →

# Headers de autenticação
x-license-key: tf_xxxxxxxxxxxx
x-store-url: minha-loja.com

# POST /scrape
{
  "url": "https://shein.com/item/..."
}

# Resposta → job_id
{ "job_id": "abc123" }
